图像分割是图像分析和模式识别的首要问题，也是图像处理的经典难题之一，它是图像分析和模式识别系统的重要组成部分，并决定图像的最终分析质量和模式识别的判别结果。基于边界的分割方法是利用不同区域间象素灰度不连续的特点检测出区域间的边缘，从而实现图像分割。
边缘检测是所有基于边界分割方法的第一步。
本文利用了经典的边缘检测算子对视频进行边缘检测，实验结果效果较为理想。
